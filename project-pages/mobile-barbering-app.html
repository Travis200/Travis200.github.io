Mobile Barbering App
March 2022
Description
One of my projects outside of university was building a cross-platform mobile barbering application. A small student start-up approached me to develop the project. For this project, I led a small team consisting of three developers including myself. This application was developed using the Flutter framework with the Dart programming language and Firebase was used for the database. The app allows clients to see the location of local barbers through the integration of a google maps plugin, so that they can then book an appointment within the app. I contributed to all elements of the app development but my focus was predominantly on the front-end development. Unfortunately midway through the project, the client discontinued the project due to financial reasons. However, I have still gained valuable experience and insight into developing software for a client from this experience. If you are an employer and would like to see the source code for this project then contact me via email, and I will be able to send it to you.
