Human Computer Interaction
2020
Description
For my contemporary topics module in 2nd year. I chose the HCI module. This highlighted to me the importance of good UX design and how to implement this. I created a digital prototype for a mobile app for food growers in Adobe XD. This module involved using iterative development, user testing and using methods such as a hierarchical task analysis, think aloud protocol, and heuristic evaluation.
